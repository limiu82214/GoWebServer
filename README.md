# GoWebServer
簡易的WebServer框架


### version
顯示基本的html，輸入網址會尋找對應的html並顯示  
http://localhost/a.html  
http://localhost/b.html  

可以透過str參數替換html內的內容
http://localhost/a.html?str=123
